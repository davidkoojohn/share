# share

> Technology sharing
-----
* install [jekyll](https://jekyllrb.com/)
```
$ gem install jekyll
```
* dev
```
$ git clone https://github.com/davidkoojohn/share.git
# then run:
$ jekyll serve
# Or
$ bundle exec jekyll server
```
* Article path
```
.
├── _config.yml   # 配置
|
├── _drafts/      # 草稿
|   └── *.md
├── _posts        # 文章
|   ├── *.md
|   └── *.md
└── *

# pre-viws /drafts/*.md
$ jekyll serve --drafts
```

> Ps : Used [NexT](http://theme-next.simpleyyt.com/).
